One-Time Setup Required

Anthropic API Key Configuration

The RAG Transcript Analyzer workflow requires an Anthropic API key to call Claude. To complete setup:

1. Open the RAG Transcript Analyzer workflow in n8n
2. Click on the "Claude API - Analyze Transcript" node
3. Under Authentication, click "Create New Credential"
4. Set Name: "x-api-key" and Value: your Anthropic API key from console.anthropic.com
5. Save the credential, then toggle the workflow to "Active"
6. The webhook URL will be: https://edenger.app.n8n.cloud/webhook/transcript-analyzer
7. Enter this URL in the Transcript Paste Interface configuration field
